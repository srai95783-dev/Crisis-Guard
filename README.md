# 🚨 CrisisGuard

## 📌 Problem Statement
Delayed emergency response and poor coordination during crises in hotels can put lives at risk.

---

## 💡 Solution
CrisisGuard is an AI-powered emergency response system designed for hotels. It detects emergencies and instantly alerts staff, guests, and emergency services for faster response.

---

## ⚙️ Features
- 🚨 SOS Emergency Button  
- 🤖 AI-based emergency detection  
- 📊 Live dashboard for staff  
- 📍 Location tracking  
- 📢 Instant alerts & notifications  
- ☁️ Cloud-based system  

---

## 🔄 How It Works
1. Emergency detected (user or AI)  
2. Data sent to cloud  
3. AI processes  
4. Alert generated  
5. Notifications sent  
6. Emergency handled  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS  
- Backend: Firebase  
- AI: Google Gemini  
- Cloud: Google Cloud  

---

## 🚀 Future Scope

- Integration with emergency services  
- Voice-based alerts
